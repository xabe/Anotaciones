Anotaciones en Java
======

El proyecto consiste un ejemplo de como crear anotaciones en java y usarlo.


Ejecución
--------------

Lo primero que tenemos es ejecutar nuestro proyecto que tiene un perfil para leventar un jetty embebido:


```sh 
git clone https://github.com/xabe/Anotaciones.git
cd Anotaciones
mvn clean package 
```

Con esto ya tenemos compilado y empaquetado nuestro proyecto ahora vamos a ejecutarlo

```sh 
java -jar target\Anotaciones.jar
```
